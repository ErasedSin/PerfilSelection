# PerfilSelection (Menu estilo jogo)

Projeto pronto para GitHub Pages.

## Como trocar links
Edite `links.json`:
- `name`: nome que aparece
- `url`: link
- `image` (opcional): caminho para imagem local (ex: `assets/instagram.png`)

Sem `image`, o site tenta pegar um ícone nítido automaticamente (Clearbit/DuckDuckGo/Google).

## Personalizar fundo
Coloque uma imagem em `assets/bg.jpg`.
